C-Theatre-Project

Console-based movie theatre system with ticketing and concessions.

Features

  • Ticketing — Pick from 5 movies, adult/child/senior, visual seating (rows A–C, 10 seats).
  • Concessions — Combos, beverages, popcorn with running total.
  • Receipt — Running total, 7% sales tax, final receipt to final_tickets.txt.

Build & run

Visual Studio: Open AI-IMAXSystemProject2Mohamed-1/AI-IMAXSystemProject2Mohamed/AI-IMAXSystemProject2Mohamed.sln, build and run.

g++:

cd AI-IMAXSystemProject2Mohamed-1/AI-IMAXSystemProject2Mohamed
g++ -std=c++11 -o Theatre AI-IMAXSystemProject2Mohamed.cpp
./Theatre   # or Theatre.exe on Windows
